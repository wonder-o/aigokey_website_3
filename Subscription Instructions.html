续费、升级和联系客服

开通日卡、周卡、到期续费、升级专业版、降级轻量版、确认额度或办理发票，都可以通过客服处理。

1. 登录查看当前订阅

先进入订阅站点查看当前使用状态、到期时间和使用习惯，再决定开通日卡、周卡、续费原档位还是调整档位。

2. 联系客服确认档位

告诉客服需要日卡、周卡、续费、升级或降级，并确认对应周期、价格、每日额度和周期总额度。

3. 开通后确认额度

完成开通、续费或变更后，建议确认每日额度和周期总额度是否与所选档位一致，避免后续使用误解。

需要续费或确认方案？

添加微信客服，说明你要开通日卡、周卡、续费、升级、降级、开票或确认额度。客服会根据你的当前订阅和使用需求，协助处理后续开通。

常见订阅问题

开通前和续费前，可以先看清楚额度、发票、变更和服务边界。

五档套餐分别是多少钱？

日卡 ¥5/天，周卡 ¥48/周；月卡包含轻量版 ¥168/月、标准版 ¥300/月、专业版 ¥588/月。所有套餐都不是 API 按次调用，也不是按量付费。

每日额度和周期总额度分别是多少？

日卡每天 $30；周卡每天 $50、周总 $3500；轻量版每天 $50、月总 $1500；标准版每天 $100、月总 $3000；专业版每天 $200、月总 $6000。

可以中途升级或降级吗？

如需从日卡、周卡切换到月卡，或升级、降级、调整订阅档位，请联系客服确认当前订阅状态、剩余周期和可用处理方式。

续费时需要提供什么？

联系客服时请说明账号、当前订阅档位、希望开通或变更的档位，以及是否需要开具发票。

可以开发票吗？

可以。AigoKey 支持按实际订阅金额开具发票，开通或续费时请联系客服提供发票抬头、税号和接收方式。
